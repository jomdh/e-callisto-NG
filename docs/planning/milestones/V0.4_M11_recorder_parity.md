# Milestone M11 — Recorder operational parity

**Closes:** v0.4.0  **Branch:** `0.4-dev`  **Theme:** the day-to-day `callisto.exe`
behaviors that deployed heterodyne stations rely on but NG lacks.

The recorder *protocol* is already faithful (M0). This milestone adds the
*operational* behaviors around it, mapped from `WINDOWS_FUNCTIONALITY.md` §1.

## Parity gaps -> deliverables

| # | Legacy behavior (evidence) | NG today | Deliverable |
| -- | -- | -- | -- |
| D1 | **Data-loss watchdog**: high-byte appears unexpectedly -> auto-stop, warn "Check RS232", auto-restart; detect receiver auto-reset "Watchdog triggered" (`mainunit.cpp:Timer1Timer`) | none | watchdog in acquisition: detect stalled/garbled stream, raise an alert, auto-stop + auto-restart; detect receiver reset |
| D2 | **Scheduler modes**: `scheduler.cfg` = `hh:mm:ss,focuscode,mode,frqfile`; at each due time start/stop, **switch frequency program**, or **trigger overview** (`Scheduler.cpp:EvalScheduler`) | start/stop windows only | extend Schedule: scheduled program-switch + scheduled overview (mode + frqfile parity) |
| D3 | **Spectral overview / "radio monitoring"**: 45-870 MHz sweep (`F0045.0`/`L13200`/`P2`), log `frequency;amplitude` to `OVS_*.prn` + `.csv` (`mainunit.cpp:OverviewClick`) | driver `overview()` exists; no trigger or OVS output | on-demand + scheduled overview run; write `OVS_*.prn`/`.csv`; surface in Tools + data browser |
| D4 | **Light curves**: daily `LCYYYYMMDD_{ADU|SFU}_<title>.txt`, up to 10 flagged channels, one "online test point" (`mainunit.cpp:SaveLightcurves`; flag `,>0` in freq file) | LC CSV (different name/format); flag not carried from program | legacy LC file format/naming; carry the per-channel light-curve flag from the frequency program; cap 10 |

## Acceptance

- A garbled/stalled serial stream auto-recovers and alerts (fake-serial test).
- A schedule can switch program and trigger an overview at set times.
- An overview run produces `OVS_*.prn`/`.csv` and is reachable from Tools.
- Light-curve files match the legacy name/format; flagged channels honored.
- Quality gate green; SNR clean; tag v0.4.0.

## Out of scope

Bench tools (M12), viewers/publication PNG (M13), SFTP/backup tree (M14).
