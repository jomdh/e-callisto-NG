# Milestone M31 — RX-888 MkII SDR driver

**Closes:** v0.8.2 area (driver)  **Branch:** `0.8-dev`  **Theme:** a real class-2
SDR plugin for the RX-888 MkII (Cypress FX3 USB3), host DSP -> normalized spectra,
so the operator's RX-888 streams through the same pipeline as a Callisto.

## Deliverables
- `drivers/sdr/rx888.Rx888Driver` implementing InstrumentDriver: USB3 IQ -> host
  FFT -> normalized 8-bit spectra; RX-888 capabilities (HF direct + VHF/UHF tuner).
- IQ-source seam: `SyntheticRx888Source` (hardware-free, runs CI + a backend-less
  Pi) and `_open_hardware_source` (lazy RX-888 backend; the real FX3 streaming
  plug-in). `identify` reports `synthetic` vs the real firmware -> provenance.
- `build_driver` routes `sdr_soft` + an RX-888 address (`usb:04b4:00f3`/`rx888`)
  to the RX-888 driver; the scan fills that address.

## Acceptance
- Contract conformance + synthetic streaming + routing tested; full record->FITS
  proven (128x10). Real FX3 streaming is a documented seam (needs the backend on
  the station). Gate green.
