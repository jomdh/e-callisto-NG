# Milestone M13 — Spectrum viewer & publication

**Closes:** v0.4.2  **Branch:** `0.4-dev`  **Theme:** the legacy visualization +
public-output tools (`M9703APlotter`, `Plott_OVS`, `wwwgeni`) and the dB units
view. Maps `WINDOWS_FUNCTIONALITY.md` §2 and design §8.1 / §6b.

## Parity gaps -> deliverables

| # | Legacy tool (evidence) | Deliverable |
| -- | -- | -- |
| D1 | **M9703APlotter / Plott_OVS**: interactive 2-col spectrum viewer; LO conversion, dB/log scaling, background subtraction, zoom, BMP export (`imagewn.cpp`, `UnitYT.cpp`) | spectrum/overview viewer island: load a `.prn`/overview, LO convert, dB/log, background subtract, zoom, PNG export |

## Acceptance

- The viewer loads an overview/spectrum, applies LO + dB/log + background, exports PNG.
- A dB toggle switches the live view and viewer without ever auto-estimating.
- The public light-curve PNG renders from a day's LC file (golden-image test).
- Live single-spectrum + LC panels update alongside the waterfall.
- Quality gate green; SNR clean; tag v0.4.2.
